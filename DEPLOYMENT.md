# Deployment Guide - Nabha Telemedicine Platform

## Overview
This guide provides step-by-step instructions for deploying the Nabha Telemedicine Platform to various hosting platforms. The application is a React-based frontend with mock backend data, optimized for rural healthcare access.

## Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Git (for version control)
- Account on deployment platform (Vercel/Netlify/etc.)
